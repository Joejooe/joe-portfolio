Mini Desktop Simulation

A desktop environment simulation built with Java and JavaFX for a university project. The system features multiple interactive applications such as Chrume (a mock web browser), Notepad, and a simple file manager for saving text and image files. Users can open preset websites like RUtube.net, RUtify.net, and stockimages.net, and use a mini photo editor to view and edit saved images. This project explores UI design, state management, and file handling using pure JavaFX.
